🔧 CBH Learn — ทดสอบ GAS Connection

ใช้หน้านี้ตรวจสอบว่า Google Apps Script URL เชื่อมต่อได้หรือไม่

⚠️ ขั้นตอนถ้าเชื่อมไม่ได้

  1. เปิด Google Sheets → Extensions → Apps Script
  2. วางโค้ดจาก gas/Code.gs ทั้งหมด → Save (Ctrl+S)
  3. Deploy → New Deployment (ต้องใช้ New ไม่ใช่ Manage)
  4. Type: Web App · Execute as: Me · Access: Anyone
  5. Authorize → Deploy → Copy URL
  6. วาง URL ในช่องด้านบน → กด Ping
  7. ถ้าผ่าน → กด Setup Sheets → แล้วกด Copy URL → วางใน js/app.js บรรทัด const GAS_URL = '...'
